## 1. Обзор продукта

### 1.1 Название
**FamilyTracker** — семейный планировщик напоминаний

### 1.2 Цель
Веб-сайт для семьи из 2 взрослых и 8 детей (3 учатся в школе), который помогает не пропускать оплату коммунальных услуг, налогов и школьных платежей.

### 1.3 Проблема
- Семья пропускает сроки оплаты налогов, ЖКХ и школьных взносов
- Нет единого места где видно все предстоящие платежи
- Непонятно кто за что отвечает

### 1.4 Решение
Простой сайт с напоминаниями, календарём и распределением ответственности по членам семьи.

## 3. Функциональные требования

### 3.1 Категории напоминаний

#### 🏠 Коммунальные услуги
- [ ] Газ — ежемесячно
- [ ] Свет (электричество) — ежемесячно
- [ ] Вай-фай (интернет) — ежемесячно

#### 🚗 Налоги
- [ ] Транспортный налог — раз в год (срок: 1 октября)

#### 🎓 Школа
- [ ] Оплата питания — ежемесячно (3 ученика)
- [ ] Родительские собрания — по расписанию
- [ ] Школьные взносы — по необходимости
- [ ] Сдача учебников / получение учебников — начало/конец года

### 3.2 Основные функции

#### F1 — Список напоминаний
- Отображение всех задач с датой и статусом
- Фильтр по категории: Налоги / Школа / ЖКХ
- Фильтр по ответственному: Папа / Мама / Ребёнок
- Статусы: 🔴 Просрочено / 🟡 Скоро (≤7 дней) / 🟢 Ок / ✅ Выполнено

#### F2 — Добавление напоминания
- Поля: название, категория, дата, ответственный, повторение
- Повторение: разово / ежемесячно / ежегодно

#### F3 — Календарь
- Вид по месяцам
- Точки на датах где есть события
- Клик на день → список событий

#### F4 — Дашборд (главная страница)
- Счётчики: просрочено / на этой неделе / в этом месяце
- Список срочных задач
- Быстрые действия

#### F5 — Члены семьи
- Профили: Папа, Мама, 8 детей
- У каждого свои задачи
- Счётчик активных задач

#### F6 — Сохранение данных
- Данные сохраняются в браузере (localStorage)
- Не теряются при закрытии страницы

## 6. Пользовательские сценарии (User Stories)

### US-01: Просмотр срочных задач
> Как **мама**, я хочу открыть сайт и сразу видеть что нужно оплатить в ближайшие дни, чтобы не пропустить срок.

**Критерии приёмки:**
- На главной странице видны все задачи со сроком ≤ 7 дней
- Просроченные выделены красным
- Есть кнопка «Отметить выполненным»

### US-02: Добавление налога
> Как **папа**, я хочу добавить транспортный налог с датой 1 октября и повторением каждый год, чтобы не забывать каждый год.

**Критерии приёмки:**
- Форма добавления с полем «повторение: ежегодно»
- После сохранения задача появляется в списке
- В следующем году задача создаётся автоматически

### US-03: Школьные платежи
> Как **мама**, я хочу видеть отдельно задачи по каждому ребёнку в школе, чтобы знать кому что нужно.

**Критерии приёмки:**
- Фильтр по ребёнку: «9 класс», «5 класс», «3 класс»
- Иконка школы у школьных задач

### US-04: Оплата ЖКХ
> Как **папа**, я хочу получать напоминание о газе, свете и вай-фай каждый месяц.

**Критерии приёмки:**
- Задачи с повторением «ежемесячно» автоматически создаются каждый месяц
- Видно сколько осталось дней
